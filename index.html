unicode-rs

Unicode support modules for Rust. All of these are available on crates.io.

unicode-normalization [doc] - This crate provides functions for normalization of Unicode strings, including Canonical and Compatible Decomposition and Recomposition, as described in Unicode Standard Annex #15.

unicode-segmentation [doc] - This crate determines Grapheme Cluster and Word boundaries according to Unicode Standard Annex #29 rules.

unicode-width [doc] - This crate determines the displayed width of char and str types according to Unicode Standard Annex #11 rules.
